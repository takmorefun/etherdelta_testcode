# etherdelta_testcode

## environment
-  node v9.11.1
-  npm 5.6.0
-  metamask(ropsten network)
-  infura(ropsten network)

## set up

### clone this repository to local environment

```
git clone https://github.com/takmorefun/etherdelta_testcode.git
```
### install node module

```
cd /path/to/your/etherdelta/directory
npm install
```
